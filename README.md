# 壹零贰肆少儿编程导航系统

## 项目简介
这是一个专为壹零贰肆少儿编程中心设计的综合导航系统，包含导航页面和积分商城两大核心功能。
- **导航页**：快速访问机构平台及编程学习资源
- **积分商城**：学生积分查询和商品兑换管理系统

### 2. 积分商城 (points-store.html)
学生积分查询和管理系统，包含以下功能：

#### 学生功能区
**积分查询**
- 输入学号和姓名查询积分
- 显示学生基本信息（学号、姓名、课程、学校）
- 显示当前积分余额

**商品浏览**
- 商品列表展示（卡片式布局）
- 分类筛选（全部、饮品、零食、玩具）
- 商品详情查看（图片、名称、积分、库存、描述）
- 库存状态显示

#### 管理员功能区
**登录方式**
- 学号：`1024000`
- 姓名：`小布老师`

**数据统计面板**
- 实时显示学生总数
- 实时显示商品总数
- 实时显示总积分
- 实时显示总库存

**数据管理**
1. **上传JSON文件**
   - 批量更新学生和商品数据
   - 自动验证数据格式
   - 支持完整数据导入

2. **导出数据**
   - 导出完整JSON数据文件
   - 包含学生、商品、课程、财务等所有信息
   - 带时间戳的文件名
   - 详细的导出统计信息

3. **同步管理系统数据**
   - 从后台管理系统 (manage.1024code.top) 获取数据
   - 三步操作流程：
     - ① 打开管理系统
     - ② 执行导出代码（自动复制到剪贴板）
     - ③ 粘贴数据并导入
   - 完整同步学生积分和商品库存
   - 同步后自动更新统计数据

4. **商品管理（完整CRUD）**
   - **添加商品**：新增积分商品到商城
   - **编辑商品**：
     - 商品名称
     - 分类（饮品/零食/玩具）
     - 所需积分
     - 库存数量（支持快捷调整 -10/-1/+1/+10）
     - 商品图片URL
     - 商品描述
   - **删除商品**：二次确认防止误删
   - **保存修改**：实时更新商品信息

**操作特点**
- 数据实时预览和统计
- 所有操作带详细的成功/错误提示
- 控制台输出完整日志
- 支持数据导入导出备份
- 响应式设计，适配各种屏幕

#### 数据安全
- 所有数据操作在本地浏览器完成
- 不经过任何服务器
- 支持随时导出备份
- 跨域数据同步通过用户手动复制粘贴

## 使用指南

### 普通用户
1. 访问导航页 (index.html)
2. 点击相应卡片跳转到目标网站
3. 点击"积分商城"查询积分和浏览商品

### 管理员
1. 进入积分商城 (points-store.html)
2. 输入管理员账号登录（学号：1024000，姓名：小布老师）
3. 使用管理功能：
   - 查看数据统计
   - 同步管理系统数据
   - 管理商品（增删改查）
   - 导出数据备份

### 数据同步流程
1. 点击"获取系统数据"
2. 打开管理系统 (manage.1024code.top)
3. 在管理系统控制台执行导出代码
4. 复制生成的JSON数据
5. 返回积分商城粘贴数据并导入
6. 同步完成后导出备份

## 部署说明

### Vercel部署
项目包含 `vercel.json` 配置文件，支持一键部署：

```json
{
  "version": 2,
  "name": "1024-navigation",
  "headers": [...],        // 安全和缓存策略
  "cleanUrls": true,       // 隐藏 .html
  "trailingSlash": false   // 无尾部斜杠
}
```

**部署步骤：**
1. 将代码推送到 GitHub
2. 在 Vercel 中导入项目
3. 自动读取 vercel.json 配置
4. 一键部署完成

**部署优化：**
- ✅ CSS/图片缓存 1 年
- ✅ HTML 不缓存（更新即时生效）
- ✅ 全球 CDN 加速
- ✅ XSS 攻击防护
- ✅ 点击劫持防护
